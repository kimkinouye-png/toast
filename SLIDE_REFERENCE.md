# Slide & Figure Reference Guide

## How to Reference Slides

Use format: **Slide XX** or **S-XX** (e.g., "Slide 12" or "S-12")

## How to Reference Figures

Use format: **Fig XX-X** where:
- First number = slide number
- Letter = figure identifier on that slide

Example: "Fig 10-A" = Figure A on Slide 10
